# 3130SEF-Project
3130SEF-Project app

Make sure you have the following installed:\
Node.js (LTS Version) : https://nodejs.org/en/download \
Ionic CLI: Run below code in your cmd
```
npm install -g @ionic/cli
```
in your terminal.
2. Setup the Project
Clone the repository and install the dependencies:
code

# Clone the project
```
git clone https://github.com/Rusher047/3130SEF-Project.git
```
# Enter the folder
```
cd 3130SEF-Project-app
```
# Install all necessary libraries (IMPORTANT)
```
npm install
```
3. Run the App
To view the app in your browser:
code
```
ionic serve
```

# When creating a new page
Run this code below:
```
# Correct way to generate a page:

ionic generate page pages/PAGE_NAME --standalone=false
```
